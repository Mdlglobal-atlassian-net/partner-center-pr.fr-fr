# <a name="microsoft-cloud-agreements-by-region-and-language"></a>Contrats Microsoft Cloud par région et langue

**S'applique à**

- Partenaires fournisseurs de solutions Cloud

Avant de pouvoir placer une commande pour le compte d’un client, le client doit accepter et signer l’accord de Cloud de Microsoft.

>[!IMPORTANT]
> À partir du 7 novembre 2018, vous devrez confirmer qu’un client a accepté le contrat de Cloud de Microsoft en fournissant des informations sur la personne qui l’a signé. Consultez [confirmer l’acceptation d’un client de l’accord de Cloud Microsoft](confirm-consent.md) pour plus d’informations.

Les contrats de Cloud de Microsoft sont spécifiques à une région et disponibles dans plusieurs langues. Rechercher l’emplacement et la langue par défaut du client dans une des listes ci-dessous pour télécharger le bon contrat.

Si vous ne savez pas les régions et les marchés, vous pouvez vendre, rechercher l’emplacement de votre entreprise dans cette liste : [Cloud marchés régionaux du programme fournisseur de solutions et des devises](regional-authorization-overview.md).
